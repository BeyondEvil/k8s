### `fn .withFsType`
Filesystem type to mount. Must be a filesystem type supported by the host operating system. Ex. "ext4", "xfs", "ntfs". Implicitly inferred to be "ext4" if unspecified.
```jsonnet
{
  withFsType(fsType):: {}
}
```
