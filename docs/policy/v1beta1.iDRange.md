
## iDRange
IDRange provides a min/max of an allowed range of IDs.

**Functions:**

[`fn .withMax`](#fn-withmax)  
[`fn .withMin`](#fn-withmin)  

---


### `fn .withMax`
max is the end of the range, inclusive.
```jsonnet
{
  withMax(max):: {}
}
```

### `fn .withMin`
min is the start of the range, inclusive.
```jsonnet
{
  withMin(min):: {}
}
```
